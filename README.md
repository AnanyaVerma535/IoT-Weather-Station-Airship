IoT Weather Station Airship 🌦️🚁

This project is an IoT-based weather monitoring system that collects real-time environmental data such as temperature, humidity, pressure, and air quality using sensors. The data is transmitted to the cloud/dashboard for monitoring.

📌 Features
- Real-time weather monitoring
- Temperature & Humidity measurement
- Air quality monitoring
- Cloud data upload (IoT dashboard)
- Live sensor visualization
 🛠 Hardware Components
- ESP32 / NodeMCU
- DHT11 / DHT22 sensor
- BMP180 / BMP280 sensor
- MQ135 gas sensor
- OLED Display (optional)
- WiFi Module

💻 Software Tools
- Arduino IDE
- ThingSpeak / Firebase / Blynk
- Embedded C / Python
